# 我的 GitHub 網站

這是一個可直接部署到 GitHub Pages 的靜態網站。

## 檔案

- `index.html`：網站首頁
- `styles.css`：網站樣式

## 發佈到 GitHub Pages

1. 在 GitHub 建立一個新的 repository。
2. 把 `index.html` 和 `styles.css` 放到 repository 根目錄。
3. 到 repository 的 `Settings` → `Pages`。
4. 在 `Build and deployment` 選擇 `Deploy from a branch`。
5. Branch 選 `main`，資料夾選 `/root`，按 `Save`。

幾分鐘後，GitHub 會提供你的網站網址。

## 可修改的地方

- 把「我的網站」改成你的名字或品牌。
- 把「作品一、作品二、作品三」換成你的專案。
- 把 GitHub、Email、LinkedIn 連結換成你的個人連結。
